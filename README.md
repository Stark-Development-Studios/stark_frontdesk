# Description
Stark Front Desk is a resource that allows for players to send notifications to either on duty law enforcement personnel, EMS, or mechanics for any non-emergency related assistance such as requesting a license or requesting job interviews. Notifications will be sent via either regular notification scripts or dispatch scripts.

## Installation
1. Add ```stark_frontdesk``` to your resources folder for your server
2. Be sure to remove the '-main' from the end of the folder name
3. Customize the script to your liking in the Config!
4. Add ```ensure stark_frontdesk``` to your server.cfg
6. Restart your server and enjoy!!

# Features
1. Targetable Locations To Send Alerts To PD, EMS, or Mechanics
2. Notification Menu Interface To Send Specific Request Types
3. Supports Either Notifications or Dispatch System Alerts
